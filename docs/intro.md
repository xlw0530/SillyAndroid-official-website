# 快速开始

欢迎使用 **SillyAndroid**！这是一款基于 React Native 和 SillyTavern（蠢酒馆）开发的安卓应用。

## 为什么选择 SillyAndroid？

- ❌ 不需要安装 Termux
- ❌ 不需要输入命令行代码
- ❌ 不需要打开浏览器访问
- ✅ 安装 APK → 更新资源 → 开始使用

## 安装步骤

下载 APK → 安装应用 → 打开等待资源更新 → 开始使用 🎉

:::info 提示
首次安装可能需要在系统设置中允许「安装未知来源应用」权限。
:::
