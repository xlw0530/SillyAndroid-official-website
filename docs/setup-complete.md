## 第一步：配置 API

:::warning 📱 小提示
- 如果你是第一次部署酒馆，你可能不知道的是酒馆的api **几乎都是付费的**哦！
- 但是有白嫖的方法如公益站，或者公益cli反代站。以及很多api贩子。市场价根据具体模型差别很大。
- 其他免费ai软件大部分用的都是本地ai或者他们自己搭的api**好用的怎么可能免费给你们免费用！**
- 就连我做这个app都是为了网盘推广，切记切记免费的才是最贵的，有问题可以来群里问我。
:::
