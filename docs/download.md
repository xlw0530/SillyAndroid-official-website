# 快速开始

欢迎使用 **SillyAndroid**！这是一款基于 React Native 和 SillyTavern（蠢酒馆）开发的安卓应用。

### 💬 官方交流

- **QQ 群**: `960130693`

## 为什么选择 SillyAndroid？

- ❌ 不需要安装 Termux
- ❌ 不需要输入命令行代码
- ❌ 不需要打开浏览器访问
- ✅ 安装 APK → 更新资源 → 开始使用

## 安装步骤

1. 下载 APK 文件。
2. 安装应用（首次安装需允许“安装未知来源应用”权限）。
3. 打开 App，等待资源自动更新。
4. 开始使用 🎉
